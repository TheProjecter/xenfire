The GTK+ toolkit was originally developed as the basis for the Gnu Image Manipulation Program, but has since been used by a wide range of applications and most notably as the foundation for the Gnome desktop environment, used by Ubuntu.

GTK+ is a cross-platform toolkit which means it can be used on several operating systems, including Windows and Mac OSX. Gaim and Gimp which are featured on this CD rely on GTK+ to run. You only need to install GTK once, which will then support both of these. You may have it installed already, in which case the installer will make this clear.
